Q1

Linked List adalah struktur data yang terdiri dari node yang saling terhubung menggunakan pointer. Berbeda dengan array, Linked List tidak menggunakan memori berurutan dan bersifat dinamis.

Kelebihan Linked List:
Ukuran fleksibel
Insert dan delete mudah

Kekurangan:
Akses lambat (harus traversal)
Butuh memori tambahan untuk pointer

Insert dilakukan dengan menambahkan node baru dan menghubungkannya ke node lain.

Delete dilakukan dengan mengubah pointer agar melewati node yang dihapus.

Stack menggunakan konsep LIFO (Last In First Out), sedangkan Queue menggunakan FIFO (First In First Out).

Operasi utama:
Stack: push, pop

Queue: enqueue, dequeue

Contoh:
Stack: undo/redo

Queue: antrian pelanggan
